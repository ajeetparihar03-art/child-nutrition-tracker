# child-nutrition-tracker
A simple personal-use web app to track what a child eats in a day using
colour-coded nutrition blocks.

## How it works
- Add food items eaten during the day
- Nutrition blocks grow vertically
- Green = good nutrition
- Orange = limit
- Red = avoid

## Tech
- Single HTML file
- Runs in Safari on iPhone
- Can be added to Home Screen like an app

## Purpose
Built for personal learning and daily use.
No calorie counting, only visual clarity.
